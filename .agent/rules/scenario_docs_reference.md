# シナリオ・キャラクタードキュメント参照ルール

シナリオの実装やキャラクターのセリフ・行動を書く際は、必ず以下のドキュメントを事前に確認してください。

## 参照すべきドキュメント

- **全体シナリオ**: `docs/integration_scenario.md`
- **個別シナリオ**: `docs/individual_scenarios/` 配下の各キャラクターファイル
  - `harutiro_scenario.md` — はるちろ
  - `massu_scenario.md` — まっすー
  - `pack_scenario.md` — ぱっく
  - `saasan_scenario.md` — さーさん
  - `tonapi_scenario.md` — となっぴー

## ルール

1. **キャラクターのセリフや反応を実装する際は、各個別シナリオの「キャラクター紹介」セクションを必ず読み、性格・特徴に一貫した言動を書くこと。**
2. シナリオ分岐やイベントを追加・変更する場合は、該当キャラクターのシナリオドキュメントおよび `docs/integration_scenario.md` との整合性を確認すること。
3. 新しいキャラクター情報が追加された場合は、対応する個別シナリオドキュメントも更新すること。
