# 追いコン用ギャルゲー 統合シナリオ案

## タイトル案

- **卒業できるの、できないの！？ 6人組最後の一年**
- サブタイトル案: 友情・進路・恋愛・研究、全部まとめて面倒を見る主人公の卒業までのコメディーADV

## 企画の軸

- プレイヤーは「友人」視点。6人グループの中心にいる、相談されがちな人物。
- 卒業までの1年間（春夏秋冬）を通して、5人の仲間の相談に乗りながら全員の卒業を目指す。
- ギャルゲー風だが、恋愛だけでなく、進路・研究・人間関係・過去エピソードが好感度と卒業判定に直結する。
- ルート次第で「立派なエンジニア」「普通に卒業」「引きこもり」「三角関係」「国際ホームレス」「海陽町で漁師」など温度差の激しい結末になる。

## 主人公設定

- 一人称は「友人」。
- 6人グループの中心人物で、全員から雑に頼られる。
- 基本スタンスは聞き役だが、選択肢次第で焚きつけたり、諭したり、変な方向に背中を押したりもできる。
- 恋愛対象そのものではなく、「相談役兼トラブルシューター」として物語を回す。

## ゲーム全体の流れ

### 1. プロローグ

- 時期は4月。いよいよ最終学年の1年が始まる。
- 6人グループで「今年一年がんばって、その上で全員卒業しよう！」と約束する。
- その直後から、5人がそれぞれ厄介な相談を持ち込んでくる。
- ここで最初の分岐として、誰の相談に最初に乗るかを選ばせる。

### 2. 共通ルート

- 共通ルートは春夏秋冬の4つのチャプターで進行する。
  - 第1章: 春（4月〜）「全員ちょっと危ないスタート」
  - 第2章: 夏（7月〜）「過去エピソード解放と夏のトラブル」
  - 第3章: 秋（10月〜）「卒研・進路の山場、秋の修羅場」
  - 第4章: 冬（12月〜）「最終追い込みと個別ルート確定」
- 各季節ごとに、5人それぞれの話題や選択肢が現れる。
- プレイヤーの選択によって、各キャラクターの「卒業力」が上下する。

### 3. 個別ルート突入

- 冬（第4章）の時点で最も好感度の高いキャラの個別ルートへ。
- ただし、はるちろルートだけは恋愛フラグの管理次第で三角関係ルートに分岐可能。
- 誰の好感度も足りない場合は「全員ぼろぼろ共通バッドルート」に進む。

### 4. 卒業判定・追いコン本番

- 最終章は追いコン当日。
- それまでの選択結果で、各キャラの進路・卒業可否・主人公との関係が決まる。
- エンディングでは、追いコンの壇上コメントや後日談スライドでオチをつける。

## 管理パラメータ

- **好感度**: 各キャラとの信頼値。個別ルートや特別会話の解放条件。
- **卒業力**: 研究・就活・生活力の総合値。一定未満だと卒業不可。
- **友情バランス**: 特定キャラだけを優遇しすぎると他のメンバーが拗ねる。三角関係や修羅場の条件にも使える。

## 共通ルート詳細

### 第1章: 春（4月〜）「全員ちょっと危ないスタート」

- 最終学年が始まり、就活や研究が本格化する春。全員の悩みが一気に噴出する。
- ぱっく: フランス人との恋愛と起業とエンジニア転向を全部成功させたがっている。
- まっすー: 遠距離恋愛と就職とカメラ、全部に気を取られて卒研の時間感覚が死んでいる。
- さーさん: 強すぎるが生活面が危うい。会社、研究、料理、車、全部に不穏要素がある。
- はるちろ: マチアプとDroidKaigiの出会いで脳内が恋愛100%になりつつある。
- となっぴー: カビと旅行とバレーに気を取られ、静かに危ない方向へ進んでいる。

### 第2章: 夏（7月〜）「過去エピソード解放と夏のトラブル」

- 各キャラの好感度イベントで過去の爆笑エピソードを回想として挿入する。
- ぱっく回想:
  - 用水路にたいやきを落として泳がせた話
  - 寮の留学生お出迎えを優先した話
  - 天文学サークルでFlutterアプリを作り始めた話
- まっすー回想:
  - ソフトボール全国
  - 卒研中間発表に盛大に遅刻
  - 遠距離恋愛相談をさーさんにしていた話
- さーさん回想:
  - 海陽町からの転校
  - 黒ポロ取得
  - ハッカソン無双
  - プリンターを終電まで占拠された去年の卒サプ事件
- はるちろ回想:
  - マチアプ用の服を買いに行き、店員をメンター扱いした話
  - 自己紹介をMarkdownで書きかけた話
  - gitリンクをプロフィールに載せようとした話
- となっぴー回想:
  - カビを培養しすぎている話
  - 河原で石を積んだ不吉な旅行話
  - バレーボール観戦で感情が大きく動く話

### 第3章: 秋（10月〜）「卒研・進路の山場、秋の修羅場」

- 共通ルートの山場。相談に乗るだけでなく、主人公が直接トラブル処理をする。
- ミニゲーム案:
  - **バイバイファイト**
    - 口論になった2人を、卒業ワードや思い出ワードでなだめる対戦風ミニゲーム。
    - 勝利すると友情バランス回復。失敗すると気まずさが残る。
  - **卒研スライド救出**
    - 制限時間内に資料の抜けや誤字を見つけるミニゲーム。
    - まっすー、さーさん、となっぴーの卒業力に影響。
  - **恋愛相談LINE返信**
    - はるちろ・ぱっくの送信文面を3択で整える。
    - 痛い文章を止められると好感度アップ。
  - **研究室片付けチャレンジ**
    - カビ・カメラ・資料・謎の私物を仕分ける。
    - となっぴーやまっすーの事故イベント回避条件。

### 第4章: 冬（12月〜）「最終追い込みと個別ルート確定」

- 最も好感度の高いキャラが主人公を呼び出し、本音を打ち明ける。
- ここで「友人として最後まで支える」か、「全員を均等に見る」かを選ばせてもよい。
- 前者は個別エンドが強くなり、後者は全員卒業のグループトゥルーに寄せやすい。

## 個別ルート案

### ぱっくルート

- テーマ: 国際恋愛、行動力、起業家気質、天才肌
- 主なイベント:
  - フランス人との恋愛相談
  - 起業と開発の両立相談
  - Suica感覚で海外改札を通って破滅しかける珍事件
  - たいやき伝説をなぜか成功のジンクスとして信じる
- 分岐:
  - 地に足のついた助言をすると、起業と開発が噛み合い「世界を見据えるエンジニア」に成長
  - 恋愛だけを煽ると、外国で美人局に遭ってホームレス化
  - 全部うまく噛み合うと、起業成功・国際恋愛成功・首席卒業の超ハッピーエンド

### まっすールート

- テーマ: 努力、広報力、恋愛不安、遅刻癖
- 主なイベント:
  - カメラに本気になりすぎて就職準備との両立に悩む
  - 遠距離恋愛への不安を打ち明ける
  - 卒研発表に遅れかけ、主人公が実働で救う
- 分岐:
  - 現実的な進路選択を支えると、東京就職で広告・広報方面へ
  - 趣味だけを全肯定すると、カメラ破壊からの人生やり直し
  - 恋愛と仕事の両立に成功すると、写真がバズってキラキラ業界エンド

### さーさんルート

- テーマ: 強者の余裕、生活の雑さ、地元性、万能感
- 主なイベント:
  - 会社立ち上げの相談
  - 車検切れ疑惑やミラー接触の不穏な話
  - 激辛大盛り・ねぎ回避・炒り卵カルボナーラなど生活力の危うさ
  - ハッカソン無双とカラオケでの無敵描写
- 分岐:
  - 主人公が現実面を支えると「最強だけど卒業もする人」になる
  - 放置すると海陽町に帰って漁師エンド
  - 全てが噛み合うと「俺の人生に失敗はなかった」首席エンド

### はるちろルート

- テーマ: 恋愛暴走、技術者らしいズレ、イベント参加、三角関係
- 主なイベント:
  - マチアプのプロフィール添削
  - 服装改造イベント
  - DroidKaigi遠征での出会い
  - 「マチアプの子とDroidKaigiの子、どっちにしよう」相談
- 分岐:
  - 主人公がブレーキ役になると、学業も恋愛も両立してハッピー
  - 恋愛を煽りすぎると、病んで引きこもり
  - 両方にいい顔をさせると三角関係ルートへ突入
- 特殊:
  - 三角関係ルートは追いコン当日に修羅場化し、ギャルゲーらしい盛り上がりを作りやすい

### となっぴールート

- テーマ: 静かな狂気、研究者気質、縁起の悪さ、推し活
- 主なイベント:
  - カビ培養と新規化学物質探索
  - 研究倫理ギリギリに見える行動を主人公が止める
  - 石積み旅行の不吉な伏線
  - バレーボール観戦でテンションが乱高下
- 分岐:
  - 丁寧に研究を支えると、化学系メーカー就職で堅実卒業
  - 雑に扱うと化学界追放エンド
  - 推しチーム優勝と研究成果が噛み合うと、ノーベル賞級の大団円

## エンディング構成

### フラグ分岐による結果決定

- 各キャラに `graduation_power`（卒業力）の数値フラグを持たせる（初期値: 3、範囲: 0〜10）。
- 共通ルートや個別ルートの選択肢でこの値が増減する。
- エンディングでは全5人の `graduation_power` を参照し、以下の3段階に分岐する:

| 値の範囲 | エンド | 概要 |
|---|---|---|
| 0〜3 | **バッドエンド** | 卒業不可。追いコンに来ない。 |
| 4〜6 | **ノーマルエンド** | 普通に卒業。可もなく不可もない。 |
| 7〜10 | **ハッピーエンド** | 首席級の大団円。 |

- 個別ルートで選んだキャラは選択肢で大きく変動する（+4 / +2 / -3）。
- 選ばなかったキャラは初期値 3 のまま → ノーマルエンドに収まる。
- **追いコン当日のエンディングでは全員の結果が順番に表示される。**

## 演出案

- 選択肢前に、グループLINE風の演出を入れるとテンポがよい。
- 過去エピソードは回想CG風の一枚絵か、誇張したナレーションで見せると笑いになる。
- 好感度上昇時はギャルゲー風の効果音や立ち絵アップを入れる。
- バッドエンドは重くしすぎず、「いや何でそうなった」の一枚絵と追いコン字幕で落とす。
- ハッピーエンドでは卒業証書授与後の後日談スライドを入れると締まりやすい。

## 実際のゲーム進行イメージ

1. 導入で5人の相談が一気に来る
2. 週ごとに誰を助けるか選ぶ
3. 会話イベントで過去エピソードが解放される
4. ミニゲームでトラブルを処理する
5. 中盤で個別ルート確定
6. 終盤で卒業判定と恋愛・進路の着地
7. 追いコン会場でエンディング発表

## この企画の強み

- 実在エピソードを強く使えるので内輪ネタとしてかなり盛り上がる
- ギャルゲー風の文法に、就活・研究・進路を混ぜることで「卒業イベントらしさ」が出る
- ルートごとの温度差が大きく、配信や観客プレイでも反応が取りやすい
- 主人公が「恋愛主人公」ではなく「相談役」なので、複数キャラを自然に回しやすい

## 実装時の補足

- まずは共通ルート4章 + 5人分の個別エンド3種で作ると規模がちょうどよい
- ミニゲームは2種類でも十分回る。優先度が高いのは「バイバイファイト」と「恋愛相談LINE返信」
- セリフを書くときは、各キャラの口調やよく言いそうなワードを本人確認して寄せると完成度が上がる
- さーさんの海陽町まわりや細かい地元ネタは、追加ヒアリングで補強するとルートがかなり強くなる
